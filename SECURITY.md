# Security Policy

## Supported Versions

Damascus UI ships as part of [Damascus OS](https://github.com/Deadmafiya/damascus).
The current development version (`master` branch) is the supported version.

## Reporting a Vulnerability

If you believe you've found a security vulnerability in Damascus UI, please report it
by opening a security advisory or an issue on the
[Damascus OS repository](https://github.com/Deadmafiya/damascus/issues). Include a
description of the affected code and, if possible, steps to reproduce.

Note: Damascus UI is a customized fork of eDEX-UI and inherits an older Electron
runtime for the v0.9 interim release. Treat it accordingly until the v1.0 rebuild lands.
